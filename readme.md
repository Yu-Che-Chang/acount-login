安裝執行：
---
1. Clone 此專案
```
https://github.com/Yu-Che-Chang/acount-login.git
```
2. 使用終端機到專案目錄
```
cd ~ /.../shorten-URL/
```
3. 安裝 npm
```
npm install
```
4. 執行 app
```
npm run start  //執行程式
```
終端機顯示Now server is hosting on http://localhost:3000
開始使用程式
